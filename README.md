# RENCI React Starter

This is like what you'd get from `create-react-app`, except it comes with RENCI branding assets.

## Assets

- RENCI logo
- RENCI logo variant
- RENCI color as CSS variables
- UNC color as CSS variables

## Builtin tooling

- Webpack 5
- Babel
- CSS
- SCSS
- PostCSS
- Source maps
- Hot module reloading
- React
- Image support
